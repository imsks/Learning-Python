## Setting up Virtual Env

### 1. Create VEnv

```
python -m venv myenv

```

### 2. Activate VEnv

```
source myenv/bin/activate

```

## Launch Jupyter notebook

```
jupyter notebook
```

### Install Packages

```
pip install -r requirements.txt
```

## Testing

### Normal

```
pytest
```

### Watch Mode

```
ptw
```
